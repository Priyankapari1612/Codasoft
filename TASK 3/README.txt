ATM Interface in Java
This ATM Interface is a Java-based console application ,the program simulates the basic operations of an ATM, allowing users to check their balance, withdraw money, deposit money, and more.

Features
User Authentication: Securely log in using a user ID and PIN.
Balance Inquiry: Check the current account balance.
Withdrawal: Withdraw money from the account.
Deposit: Deposit money into the account.
Transaction History: View a summary of past transactions.
Exit: Safely exit the ATM system.

Requirements
Java 8 or higher
Any Java IDE (Eclipse, IntelliJ IDEA) or command-line environment.

Usage
1.Select one of the following options from the main menu:
  Balance Inquiry
  Withdrawal
  Deposit
  Transaction History
  Exit
2.Follow the on-screen instructions to complete each action.
