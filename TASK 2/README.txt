The program calculates the total marks, average, and final grade of a student based on input scores for multiple subjects.

Features
Input scores for multiple subjects.
Calculate total and average marks.
Assign a grade based on predefined score ranges.
Simple and intuitive user interface (console-based).

Requirements
Java 8 or higher
Any Java IDE (Eclipse, IntelliJ IDEA) or a command-line environment.

Grading System
A: 90-100
B: 80-89
C: 70-79
D: 60-69
F: Below 60
