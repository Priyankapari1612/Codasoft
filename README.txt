Number Guessing Game
Overview
This project is a simple Number Guessing Game,the game randomly generates a number, and the player tries to guess it, with feedback provided on each attempt.

Game Features
Random Number Generation: A new number is generated every time the game starts.
Interactive Gameplay: The player receives feedback on whether their guess is too high, too low, or correct.
Replay Option: After guessing correctly, the player can choose to play again.
Input Validation: Ensures valid user input during the game.

How It Works
The program randomly selects a number within a predefined range (e.g., 1 to 100).
The player enters their guess.
The game provides feedback—whether the guess is too high or too low—until the correct number is guessed.
After guessing the number, the player can choose to start a new game.

Technologies Used
Language: Java
Development Environment: Any Java IDE or command-line interface

Future Enhancements
Adding difficulty levels.
Keeping track of the number of attempts.
Adding a leaderboard feature for competitive play.
